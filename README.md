# 🏠 NestVerify - Rental House Finder with Verification System

NestVerify is a web-based rental house finder platform designed to help users find safe, authentic, and verified rental properties. The system reduces fraudulent listings by implementing an admin-controlled verification process for property owners and their properties.

## 📌 Features

### For Tenants
- User Registration and Login
- Search Rental Houses
- Filter Properties by:
  - Location
  - Budget
  - Facilities
- View Verified Properties
- Contact Property Owners

### For Property Owners
- Register and Login
- Add Property Listings
- Upload Property Images
- Submit Identity and Property Documents
- Track Verification Status

### For Admin
- Review Submitted Documents
- Verify Property Owners
- Approve or Reject Listings
- Manage Property Records

## 🎯 Project Objectives

- Reduce fake property listings
- Improve trust between tenants and owners
- Provide secure property verification
- Simplify house searching
- Maintain organized property records
- Ensure transparency in the rental process

## 🔄 System Workflow

1. Property Owner Registers
2. Owner Uploads Property Details
3. Owner Submits Verification Documents
4. Admin Reviews Documents
5. Property Receives Verified Badge
6. Users Search and Filter Properties
7. Users Contact Owners for Rental Inquiries
